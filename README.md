# HackUPC-smadex

## Structure
- `/frontend`: React application (Vite)
- `/backend`: Python AI Engine logic
- `/data`: Source assets and metadata

## Getting Started
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend (Data Conversion)
```bash
python backend/convert_data.py
```
